# Summary

This is a model and OpenSCAD file for a mold used to make silicone SimpleSumo robot mini-sumo tires. 

 The stock tires on the SimpleSumo kits can be improved upon by making your own out of a higher friction material.  I made my own using silicone molding putty and a diy version of Sugru, both worked well though the oogoo experience was messy.

Option 1: Silicone Molding Putty: http://amzn.to/2y0jRot
Option 2: Sugru: http://amzn.to/2zjqKoH
Option 3: Oogoo: https://www.instructables.com/id/How-To-Make-Your-Own-Sugru-Substitute/

Video of the experiment
https://youtu.be/ZwFOcSzwNsk
